## Phase 1 Briefs

### B-01: What Is a Limited Edition Print?

Search intent:

- Primary keyword: `what is a limited edition print`
- Secondary keywords: `limited edition art print meaning`, `limited edition print value`, `edition size art`
- Funnel stage: Learn / Trust / Compare

Street Collector relevance:

- Category page: `/limited-edition-street-art-prints`
- Shop/product link: `/shop/products`
- Supporting category: `/urban-art-prints`
- Product fact to verify: which products show edition size and whether "typically 44 editions" applies to selected examples.

Verified local claims:

| Claim | Source |
| --- | --- |
| Street Collector works with editioned artworks and COA language. | `content/shop-faq.ts` |
| Current FAQ says artworks are released in strictly limited quantity, typically 44 editions per design. | `content/shop-faq.ts` |
| Street Collector has 100+ artists language in current landing content. | `content/street-collector.ts` |

TBC before drafting:

| Claim | Needed source |
| --- | --- |
| Specific edition size for any named artwork. | Live product data / Shopify product metafield. |
| Whether every example product includes COA. | Product-level COA field or approved policy language. |
| Whether the edition number is visibly shown on the product page. | Live PDP check. |

Outline:

1. Answer-first definition of limited edition print.
2. Explain edition size, numbered copies, and why finite quantity matters.
3. Use Street Collector's typical 44-edition model as a worked example only after product verification.
4. Explain what edition size does and does not prove.
5. Link to current artworks and artist pages.
6. FAQ: meaning, value, open vs limited edition, authenticity.

Internal links:

- `/limited-edition-street-art-prints`
- `/shop/products`
- `/shop/explore-artists`
- `/shop/blog/what-is-a-certificate-of-authenticity-for-art` when the final guide exists.

Brand risk:

- Do not imply investment returns.
- Do not say every Street Collector print has the same edition size unless verified.
- Do not make edition size the only value factor; phrase it as a core scarcity mechanic.
