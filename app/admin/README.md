# Admin Portal Documentation

## Overview
The Admin Portal provides a comprehensive interface for managing all aspects of the COA Service application. This documentation covers all major sections and features available in the admin interface.

## Main Sections

### [Dashboard](/admin/dashboard/README.md)
- Overview of key metrics and system status
- Quick access to important functions
- Recent activity monitoring

### [Orders Management](/admin/orders/README.md)
- View and manage all orders
- Order status tracking
- Edition number assignment
- Order details and history

### [Product Editions](/admin/product-editions/README.md)
- Manage product editions
- Edition number assignment
- Status management (active/removed)
- Edition resequencing

### [Vendor Management](/admin/vendors/README.md)
- Vendor profiles and information
- Vendor performance metrics
- Vendor settings and configurations

### [Certificates](/admin/certificates/README.md)
- Certificate generation and management
- Certificate templates
- Certificate status tracking

### [Reports](/admin/reports/README.md)
- Sales reports
- Edition reports
- Vendor performance reports
- Custom report generation

### [Settings](/admin/settings/README.md)
- System configuration
- User management
- API settings
- Integration settings

### [Sync Management](/admin/shopify-sync/README.md)
- Shopify data synchronization
- Product synchronization
- Order synchronization
- Manual sync triggers

### [Tax Reporting](/admin/tax-reporting/README.md)
- Tax report generation
- Tax data management
- Tax settings configuration

## Access Control
- Role-based access control
