# Moshe Gilboa

Moshe Gilboa is an Israeli artist and designer whose practice doesn't file neatly. (moshegilboa)

## Overview

Moshe Gilboa is an Israeli artist and designer whose practice doesn't file neatly. Animation, UI, indie games, and illustrated essays share the same humor — work that winks while it teaches. He documents shipped projects in long-form on Medium, which tells you something about how seriously he takes the process of making.
